## 1. Font Strategy

To combine structural readability with a modern feel, the system uses two Google Fonts:

```
Font Family Allocation:
├── Header Font (Outfit): Geometric Sans-Serif. Used for Page Titles, Headings, and Numbers (GPAs).
└── UI & Body Font (Inter): Highly readable Sans-Serif. Used for Labels, Input Fields, Data Tables, and Body Text.
```

---

## 3. Font Weights & Application Rules

1.  **Bold (700)**: Used sparingly. Reserved for primary GPA numbers, page headers, and primary warning highlights.
2.  **Semi-Bold (600)**: Used for card titles, active tab labels, and primary buttons.
3.  **Medium (500)**: Used for input field labels, table headers, and navigation link text.
4.  **Regular (400)**: Used for body text, chat messages, and data table cell content.
