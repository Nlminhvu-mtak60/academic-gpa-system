---

## 1. Document Purpose

This document defines user experience layouts and call-to-actions (CTAs) for empty states across the application.

---

## 2. Empty State Layout Designs

---

### 2.1 No Academic Years / Semesters Created Yet
Displayed when a newly registered student navigates to the Semesters page.

*   **Illustration**: A folder icon with an add badge.
*   **Copy (Bilingual)**:
    *   *Vietnamese*: "Chưa có năm học nào. Hãy thêm năm học đầu tiên để bắt đầu theo dõi điểm."
    *   *English*: "No academic years yet. Create your first academic year to start tracking grades."
*   **Call-to-Action**: A primary button labeled "+ Add Academic Year" that opens the creation modal.

---

### 2.2 No Courses Added to Semester
Displayed when a student opens a newly created semester card.

*   **Illustration**: An open book outline with a plus badge.
*   **Copy (Bilingual)**:
    *   *Vietnamese*: "Học kỳ này chưa có môn học nào."
    *   *English*: "No courses in this semester yet."
*   **Call-to-Action**: A primary button labeled "+ Add Course" inside the empty table row.

---

### 2.3 No Active Goals Defined
Displayed in the Goal Planner module when no cumulative target has been set.

*   **Illustration**: A target board with an arrow missing.
*   **Copy (Bilingual)**:
    *   *Vietnamese*: "Chưa thiết lập mục tiêu GPA. Hãy đặt mục tiêu để lên kế hoạch học tập."
    *   *English*: "No active GPA goal. Define a target cumulative GPA to plan your academic roadmap."
*   **Call-to-Action**: A primary button labeled "Set Target GPA".

---

### 2.4 No AI Conversations Started
Displayed when opening the AI Advisor chat panel for the first time.

*   **Illustration**: A robot bubble outline.
*   **Copy (Bilingual)**:
    *   *Vietnamese*: "Bắt đầu cuộc trò chuyện mới với Cố vấn học tập AI."
    *   *English*: "Start a new conversation with your AI Academic Advisor."
*   **Call-to-Action**: A primary button labeled "New Chat Session".

---

### 2.5 Empty Notification Inbox
Displayed when the user has deleted or read all notifications.

*   **Illustration**: An envelope with a checkmark.
*   **Copy (Bilingual)**:
    *   *Vietnamese*: "Hộp thư thông báo trống."
    *   *English*: "You're all caught up! No notifications in your inbox."
*   **Call-to-Action**: None required. Displays a secondary return link: "Back to Dashboard".

---
