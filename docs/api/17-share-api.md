## 1. Share Endpoints Overview

These endpoints allow students to generate, list, and revoke secure public links for sharing their academic transcripts. It also provides the public access endpoint for viewing shared transcripts.

---

## 2. Endpoint Specifications

---

### 2.1 POST /transcripts/share
Generates a new public sharing link.

*   **HTTP Method**: `POST`
*   **Authorization**: Bearer token (Student role)
*   **Request Body**:
    ```json
    {
      "expiresAt": "2026-07-24T13:30:00Z"
    }
    ```
*   **Validation Rules**:
    *   `expiresAt`: Optional, must be a future datetime. If omitted, the link does not expire.
*   **Business Rules**:
    *   **Maximum Active Links**: A student is limited to a maximum of 10 active share links. If they attempt to create an 11th link, the server returns a `422 Unprocessable Entity` response.
*   **Success Response** (`201 Created`):
    ```json
    {
      "success": true,
      "message": "Share link generated successfully.",
      "data": {
        "id": "s72fb33f-8461-460d-85fa-7c961e67fa8b",
        "shareToken": "f3a8b2c4-84d2-45e0-b999-0e86b240ff3f",
        "shareUrl": "https://gpa.app/shared/f3a8b2c4-84d2-45e0-b999-0e86b240ff3f",
        "expiresAt": "2026-07-24T13:30:00Z"
      }
    }
    ```

---

### 2.2 DELETE /transcripts/share/{id}
Revokes a public sharing link.

*   **HTTP Method**: `DELETE`
*   **Authorization**: Bearer token (Student role)
*   **Path Parameters**:
    *   `id` (guid): Target share link record ID.
*   **Success Response** (`204 No Content`): No response body.
*   **Business Logic**: Immediately invalidates the sharing link, blocking public access.

---

### 2.3 GET /transcripts/shares
Lists all active sharing links generated by the student.

*   **HTTP Method**: `GET`
*   **Authorization**: Bearer token (Student role)
*   **Success Response** (`200 OK`):
    ```json
    {
      "success": true,
      "data": [
        {
          "id": "s72fb33f-8461-460d-85fa-7c961e67fa8b",
          "shareToken": "f3a8b2c4-84d2-45e0-b999-0e86b240ff3f",
          "shareUrl": "https://gpa.app/shared/f3a8b2c4-84d2-45e0-b999-0e86b240ff3f",
          "viewCount": 15,
          "expiresAt": "2026-07-24T13:30:00Z",
          "createdAt": "2026-06-24T13:30:00Z"
        }
      ]
    }
    ```

---

### 2.4 GET /shared/{token}
Retrieves an anonymized academic transcript using a public share token.

*   **HTTP Method**: `GET`
*   **Authorization**: None (Public access)
*   **Path Parameters**:
    *   `token` (guid): The `shareToken` generated by the student.
*   **Business Logic**:
    *   Validates that the token exists, is not revoked, and is not expired.
    *   **Anonymization**: Replaces the student's name with initials and hides sensitive details like emails and avatars.
    *   Increments the `viewCount` of the matching share link.
*   **Success Response** (`200 OK`):
    ```json
    {
      "success": true,
      "data": {
        "studentInitials": "N.V.A",
        "universityName": "University of Technology",
        "majorName": "Computer Science",
        "cumulativeGpa10": 7.58,
        "cumulativeGpa4": 3.12,
        "totalCreditsCompleted": 45,
        "academicRecords": [
          {
            "yearName": "2024-2025",
            "semesters": [
              {
                "semesterName": "Semester 1",
                "semesterGpa10": 8.03,
                "courses": [
                  {
                    "courseCode": "CS101",
                    "courseName": "Intro to Computer Science",
                    "credits": 3,
                    "courseScore": 8.5,
                    "letterGrade": "A"
                  }
                ]
              }
            ]
          }
        ]
      }
    }
    ```
*   **Error Responses**:
    *   `404 Not Found`: Share token does not exist, is revoked, or is expired.
