## Document Conventions

- **Priority Levels**: P0 (Must Have), P1 (Should Have), P2 (Nice to Have), P3 (Future)
- **Requirement IDs**: `FR-XXX` (Functional), `NFR-XXX` (Non-Functional), `BR-XXX` (Business Rule)
- **User Story Format**: "As a [role], I want [action], so that [benefit]"
- **Status Tags**: ✅ Approved, 🔄 In Review, ❌ Rejected, ⏳ Pending
